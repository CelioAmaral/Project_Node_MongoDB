### How to Run

1. Install Node.js and download MongoDB Community Server
2. MongoDB Compass(data visualization)
3. run a mongod server at port 27017
4. cd crud-mongodb
5. npm install
6. npm start

```bash
# Access browser url:
Web Application: http://localhost:3000
```
